Semaphore

Semaphore（信号量）是用来控制同时访问特定资源的线程数量，它通过协调各个线程，以保证合理的使用公共资源。

Semaphore 有 Lock 不易实现的功能：Semaphore 可以允许多个线程访问一个临界区。

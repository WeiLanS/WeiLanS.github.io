Linux性能优化-内存篇
1. 内存映射虚拟内存： Linux 内核给每个进程都提供了一个独立的虚拟地址空间，并且这个地址空间是连续的。进程可以很方便地访问虚拟内存。 虚拟地址空间分为：内核空间和用户空间。不同字长处理器（32位/64位）地址空间范围也不同。进程在用户态时，只能访问用户空间内存；只有进入内核态后，才可以访问内核空间内存。 所有进程的虚拟内存加起来要比实际内存大得多，因此不是所有虚拟内存都分配内存，只有实际使用的虚拟内存才会分配物理内存，分配后的物理内存通过内存映射进行管理。内存映射，其实就是将虚拟内存地址映射到物理内存地址。为了完成内存映射，内核为每个进程都维护了一张页表，记录虚拟地址与物理地址的...
